# Changelog

## [1.1.0] - 2026-04-13

### Added - 2026-04-13

- **SQLite Database** for historical data storage
- **Statistics Page** (`/stats`) with interactive charts (Chart.js)
- **Automatic storage** of weather data on every search
- **Historical charts**: temperature (max/avg/min), humidity, pressure, wind
- **Hourly trend** of last 24 hours with dual axis
- **Summary cards** with synthetic statistics
- **Multi-language support IT/EN** with language selector in menu
- **Complete internationalization system** (i18n)
- Period selector: 7, 30, 90 days or 1 year
- Navigation menu between Home and Statistics
- API endpoints for statistical data (`/api/stats/*`)
- Automatic cleanup of old data (configurable)

### Changed

- Extended ADM compatibility from 2.0+ (previously 4.0+)
- Removed screenshot requirement from `apkg.xml` for ADM 2.0
- APK structure: files split into CONTROL/ and data/ (standard ASUSTOR format)
- Added enable/disable icons for ADM desktop visualization
- Automatic installation with app start at completion
- Improved install.sh script for file copy to app root

## [1.0.0] - 2026-04-10

### Added - 2026-04-10

- Initial release of Weather App for ASUSTOR NAS
- **Migration to Open-Meteo API** - No API key required
- Current weather support with Italian descriptions
- 5-day forecasts with min/max temperatures
- Italian interface
- Integrated weather SVG icons with modern design
- Support for any city worldwide
- User preferences saving (city) in localStorage
- Multi-language support (Italian/English)
- ASUSTOR APK package compatible with App Central
- Automated installation, start and stop scripts
- Complete documentation in Italian

### Technical Features

- Python Flask backend
- HTML5, CSS3, vanilla JavaScript frontend
- RESTful API for weather data
- Compatibility with ASUSTOR ADM 2.0+
- Configurable port 8000
- Integrated SQLite database
